Frequently Asked Questions

1. Is Meal Prep Codex free to use?

Yes! The platform is completely free to use. You don’t need to sign up or subscribe—just browse recipes and enjoy.

2. How do I submit a recipe?

Currently, recipe submissions are not available. However, I’m working on a feature that will allow users to contribute original recipes.

3. Can I use the Meal Prep Codex API?

Yes! This project includes an API that provides a large collection of recipes. If you're a developer and want to integrate the API into your own app, feel free to contact me for more details.

4. Are the recipes original?

Some recipes are sourced using web scraping techniques, but I also focus on providing unique and original dishes. My goal is to create a collection of diverse and accessible recipes.

5. Can I request a specific recipe?

At the moment, I don’t have a request feature, but I’m open to suggestions! Feel free to reach out if there’s a dish you’d like to see added.

6. What technology powers this platform?

Meal Prep Codex is built using Django for the backend, with APIs and Web Scraping to provide a wide variety of recipes. The project started as a learning experience and evolved into a useful tool for recipe lovers.

7. Who created Meal Prep Codex?

This is a solo project, developed out of a passion for coding and cooking. I built it as a way to learn modern web technologies while providing a useful free service.

8. How can I report a bug or give feedback?

Since this is a one-person project, I really appreciate feedback! If you find any issues or have suggestions, please contact me.
